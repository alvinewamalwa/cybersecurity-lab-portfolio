# Network Reconnaissance using Nmap

## Objective
To identify open ports and running services on a target system.

## Tools Used
- nmap

## Process
- Performed basic scan using `nmap <target>`
- Identified open ports and services
- Reviewed service versions where available

## Observations / Findings
- Open ports may expose services to potential attacks
- Unnecessary open ports increase attack surface

## Security Insight
Regular network scanning helps identify vulnerabilities before attackers exploit them.
