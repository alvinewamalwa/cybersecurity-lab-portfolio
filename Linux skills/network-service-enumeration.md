# Network Service Enumeration Using Nmap

## Task
Identify open ports and services running on a target system.

## Commands Used
- nmap

## Execution
nmap -sV 192.168.1.1

## Result
Open ports and service versions were identified, exposing running services on the target system.

## Security Insight
Service enumeration helps identify exposed attack surfaces that could be exploited if not secured or updated.
